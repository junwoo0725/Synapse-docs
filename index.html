Synapse-docs

로컬 LLM 기반 AI 문서 분석 도구

"안정적이고 확장 가능한 인프라를 구축해 서비스의 성능과 효율을 극대화하는 클라우드 개발자"

프로젝트 개요

기존 문서 요약 도구는 명확한 한계를 가지고 있었습니다.

🚫

1. 인식 한계

텍스트만 인식 가능해 이미지 기반 자료 (PDF 스캔본)를 처리하지 못했습니다.

💰

2. 높은 비용

OpenAI 등 외부 API 사용으로 인한 높은 비용이 지속적으로 발생했습니다.

🔒

3. 보안 문제

로컬 작동이 불가능해, 민감한 문서를 외부에 전송해야 하는 보안 이슈가 있었습니다.

솔루션: Synapse Docs

이러한 문제를 해결하기 위해, OCR(이미지 문자인식)과 로컬 LLM(Ollama)을 결합하여 네트워크 연결 없이도(완전 오프라인) 작동하는 AI 문서 분석 도구를 개발했습니다. PDF/텍스트 파일을 업로드하면 요약, 키워드 추출, 비교, Q&A까지 자동으로 수행합니다.

핵심 기능

Synapse-docs는 두 가지 강력한 핵심 기능을 제공합니다. 아래 탭을 클릭하여 확인해 보세요.

문서 자동 요약 및 키워드 추출

사용자가 PDF/TXT 파일을 업로드하면, 텍스트를 페이지별로 분할하고 전처리합니다. 이후 선택된 LLM(Gemini 또는 Ollama)을 통해 핵심 요약과 주요 키워드를 자동으로 생성합니다.

  • 페이지별 텍스트 분할 및 자동 전처리
  • Gemini 또는 Ollama 기반 핵심 요약 생성
  • 주요 키워드 자동 추출
  • Streamlit 인터페이스에서 요약 수준 (짧게/기본/자세히) 조정 가능

로컬 Q&A 및 문서 맥락 질의응답

Ollama 임베딩 모델(nomic-embed-text)을 사용해 문서 조각을 벡터화합니다. 사용자의 질문 역시 임베딩하여 코사인 유사도 기반으로 문서 내 관련 내용을 탐색(Top-K)합니다.

  • Ollama (nomic-embed-text) 기반 문서 벡터화
  • Numpy 코사인 유사도 기반 Top-K 조각 탐색
  • 탐색된 컨텍스트 기반으로 정확한 Q&A 수행
  • 시스템 프롬프트 강화를 통해 "문서에 없는 정보는 모른다"고 답변 (환각 방지)
  • 완전 오프라인 환경에서도 질문 가능
